# Archived Planning Documents

This directory contains **historical planning documents** from Morphogen's development. These are preserved for reference but are no longer actively maintained.

## Purpose

These documents represent:
- **Past strategic planning** (Q4 2025 roadmaps, release plans)
- **Completed analyses** (domain value analysis, capability frameworks)
- **Evolution strategies** (architectural roadmaps, selective evolution)
- **Implementation tracking** (domain progress logs)

They capture the thinking and planning from specific points in time but have been **superseded by the unified roadmap**.
